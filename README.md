# nodeJs

---------
## 简易登录注册系统  
**注册成功** -- “恭喜，注册成功”  
**注册重复** -- “该用户名已被注册”  
**登录成功** -- “登录成功”  
**登录名不存在** -- “该用户仍为被注册”  
**密码不符合** -- “用户名或密码错误”  
**发生错误** -- “注册失败”/登录失败  

-----------

## 共享网盘

### 工具
 * Express
 * MySql
 * Jquery（仅基本的使用）
 * Bootstrap (较简易)
### 功能
 * 用户个人账号注册、登录
 * 用户上传文件及删除
 * 用户在网盘主页下载文件，对应下载次数更新
 * 对应数据库的增删查改
